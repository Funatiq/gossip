# ```gossip```: Efficient Communication Primitives for Multi-GPU Systems

## Build
Compile using the provided ```Makefile``` by calling ```git submodule update --init && make```.

You will need:
    
    - CUDA >= 9.2
    - GNU g++ >= 6.3.0 and compatible with your CUDA version
    - Python >= 3.0 with
        - Matplotlib
        - NumPy
